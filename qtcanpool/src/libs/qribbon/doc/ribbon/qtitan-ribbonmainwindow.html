RibbonBar * RibbonMainWindow::ribbonBar () const

Returns a pointer to RibbonBar object. If RibbonBar does not exist, then the function will create an empty Ribbon UI panel.

See also setRibbonBar().

void RibbonMainWindow::setRibbonBar ( RibbonBar * ribbonBar )

Sets the RibbonBar to the main window. Object RibbonMainWindow takes ownership of the ribbonBar. The method is used when needed to set the custom Ribbon Bar object, whose class, inherited from RibbonBar.

See also ribbonBar().
